Royal Rope is a 2-player castle-climb obby where players are chained together. Every jump, pull, and rescue moment is shared. The core fantasy is simple: survive the climb by mastering each other.

Game

A castle climb built around shared motion.

What it is

Royal Rope is a tense, funny co-op platformer where the chain between players is the main mechanic, not a penalty. Every route, jump, and recovery sequence is shaped by the rope.

What makes it stand out

The game creates immediate stories. A safe climb can become a rescue, a swing can become a save, and a mistake can still turn into a heroic recovery if both players react fast enough.

Mechanics

The rope changes how players think about movement.

Shared momentum

Players can’t separate cleanly, so movement choices affect both bodies at once.

Tension and swing

The rope acts like a moving tool, letting players build speed and bridge gaps together.

Rescue first design

Failures are recoverable, which keeps the pace energetic and the co-op fantasy intact.
